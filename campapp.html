Camp app

This app is my first individual project. This is a booking platform for nature lover.

Overview

The camping app makes possible not only to search for camping places nearby, but to book for this places and as a returning user to use collected discounts for the next booking. In the user profile the logged in user can follow the history of the past bookings and edit the current bookings. To create a project from the beginning was really challenging. First I created wireframes in Balsamiq, to have a clear overview.

The visual design was the most difficult part of this project. I decided to use the illustration from the first screen throughout the app. But to keep the usability with the design in balance was really challenging.
