Discover Austria

This is my first individual project for web, where I focused on the above the fold experience.

Overview

I have created a homepage for a climbing service Discover Austria, where the user can search for tours with a map, or booking organized tours with guiders. The colorful header is eye-catching, to compensate this colors and keep the design clear was the big challenge of this project.

The outline icons give more pureness to the page and help the user by the easier orientation on the site. This project was originally only for webview created, but as a challenge I have decided to create design for mobile as well.

Zsofia Burgundi - UI / UX Designer
